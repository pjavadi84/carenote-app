# Outreach batches

This directory holds in-progress LinkedIn DM drafts generated by the
[`/lead-batch`](../../../.claude/skills/lead-batch/SKILL.md) Claude
Code skill. Each batch is a Markdown file with one section per lead:
profile link, chosen template, personalisation hook, and the
80-word draft message ready to copy-paste into LinkedIn.

## What this is NOT

- **Not automation that sends to LinkedIn.** LinkedIn's User Agreement
  forbids automated messaging and enforces it with permanent account
  bans. The skill drafts only. You hit send yourself.
- **Not mass-scraping.** The skill attempts at most one `WebFetch` per
  lead in your input; it does not enumerate connections, mutual contacts,
  or anyone you didn't list.
- **Not a tracker / CRM.** The batch file is a write-once review
  artifact; the source of truth for "who I've sent to" stays in your
  separate tracker (the column schema is documented in
  [linkedin-cold-outreach.md](../linkedin-cold-outreach.md)).

## Privacy: batches stay local

Files in this directory are **gitignored** because they contain unsent
draft messages addressed to specific real people. Don't commit batches
even if they look harmless — the lead's name + facility + your draft
message is a PII pattern not worth pushing to a public repo.

If you want to commit one batch as documentation (e.g., for a case
study or sales-coaching session), redact the names + URLs first and
explicitly `git add -f` only that specific file.

## Quick start

In a Claude Code session inside this repo:

```
/lead-batch
```

The skill will ask which input format you have. The fastest format is
the pipe-separated inline block:

```
Name | Profile URL | Facility | City | Signal
Jane Doe | https://www.linkedin.com/in/janedoe | Sunrise Care | San Diego | Posted last week about state audits
```

Paste 5–15 leads at a time. Drafts appear in a new file under this
directory within seconds.

## Refining a single draft

If one draft doesn't land — wrong template, missed the hook, sounds
off — tell the skill which lead number and what to change. It will
redraft just that row in-place rather than starting a new batch.
